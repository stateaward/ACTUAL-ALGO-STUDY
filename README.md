## Convention

**COMMIT RULE**

커밋시, 파일명은 `문제번호_문제명_이름` 순으로 한다.

|     사이트명     |             예시             |
| :--------------: | :--------------------------: |
|     **백준**     |    `BOJ1234_탐색_고길동`     |
|     **SWEA**     |     `SW1234_탐색_김둘리`     |
| **프로그래머스** | `PG_문자열탐색하기_김도우너` |
|      **JO**      |     `JO1234_탐색_마이콜`     |

<br><br>
**COMMIT MESSAGE**

커밋메세지만 보고도 어떤 상태인지를 `직관적으로` 파악하기 위해 다음과 같은 규칙을 지향합니다.

`푼사람 / 문제명 / 특이사항`

_예시)_

```

```
